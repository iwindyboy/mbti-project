# 브라우저 콘솔 확인 방법

## 방법 1: 키보드 단축키 (가장 빠름)

### Chrome, Edge, Firefox
- **Windows/Linux**: `F12` 키 누르기
- 또는 `Ctrl + Shift + I` (또는 `Ctrl + Shift + J`)

### Mac
- `Cmd + Option + I` (또는 `Cmd + Option + J`)

## 방법 2: 마우스로 열기

### Chrome/Edge
1. 페이지에서 **우클릭** (마우스 오른쪽 버튼)
2. 메뉴에서 **"검사"** 또는 **"Inspect"** 클릭
3. 개발자 도구 창이 열림

### 또는
1. 브라우저 상단 메뉴에서:
   - **Chrome**: 메뉴(☰) → 도구 더보기 → 개발자 도구
   - **Edge**: 메뉴(⋯) → 도구 더보기 → 개발자 도구

## 방법 3: 콘솔 탭 찾기

개발자 도구가 열리면:
1. 상단에 여러 탭이 보입니다: **Elements**, **Console**, **Sources**, **Network** 등
2. **Console** 탭을 클릭합니다
3. 여기에 모든 로그 메시지가 표시됩니다

## 확인할 메시지

매칭 유형 링크 관련해서 다음 메시지들을 찾아보세요:

### 정상 작동 시
- 아무 메시지도 없거나
- 일반적인 React 로그만 보임

### 문제가 있을 때
- 🔴 **"유형 코드를 찾을 수 없습니다. 라인: ..."**
  → 정규식이 매칭하지 못함
  
- 🟡 **"유형 코드를 찾았지만 SCAN_TYPE_DETAILS에 없습니다: ..."**
  → 데이터에 해당 유형이 없음

## 콘솔에서 직접 테스트하기

콘솔 탭에서 다음 코드를 입력하고 Enter를 누르세요:

```javascript
// 정규식 테스트
const testLine = "ENTJD [든든한 방패] : 내 전략을 큰 그림에서 인정함";
const match = testLine.match(/([A-Z]{5})(?=\s|\[|$)/);
console.log('매칭 결과:', match);
console.log('찾은 유형 코드:', match ? match[1] : '없음');
```

결과가 나오면 알려주세요!

## 스크린샷 찍는 방법

문제를 더 쉽게 파악하려면:
1. 개발자 도구가 열린 상태에서
2. `Windows + Shift + S` (Windows) 또는 `Cmd + Shift + 4` (Mac)
3. 콘솔 부분을 선택해서 스크린샷 찍기
4. 스크린샷을 공유해주시면 더 정확히 도와드릴 수 있습니다!
