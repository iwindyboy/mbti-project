# SCAN 성향검사 앱 - 다음 단계 작업 목록 (업데이트)

## ✅ 완료된 작업
1. ✅ 30개 문항 데이터 반영 (5개 그룹, 각 6문항)
2. ✅ 그룹별 페이지 구조 (5개 그룹)
3. ✅ 프로그레스 바 (1/5 단계)
4. ✅ 한 페이지에 6개 문항 세로 배치
5. ✅ 이전/다음 버튼 (첫 페이지는 이전 버튼 없음)
6. ✅ 부드러운 스크롤 (답변 시 다음 문항으로)
7. ✅ 유효성 검사 (6개 모두 답변해야 다음 버튼 활성화)
8. ✅ 반응형 레이아웃 (모바일/태블릿 대응)
9. ✅ 결과 페이지 (유형 코드, 네이밍, 막대 그래프, 결제 팝업)

---

## 🔴 긴급 (즉시 해야 할 작업)

### 1. **실제 테스트 및 버그 수정** (1-2시간)
```bash
npm install
npm run dev
```

**테스트 항목:**
- [ ] 5개 그룹 모두 순서대로 진행되는지
- [ ] 각 그룹 6개 문항 모두 답변 가능한지
- [ ] 답변 시 스크롤이 제대로 작동하는지
- [ ] 프로그레스 바가 정확히 업데이트되는지
- [ ] 결과 계산이 정확한지 (5개 축 점수)
- [ ] 결과 페이지가 제대로 표시되는지
- [ ] 이전/다음 버튼이 정상 작동하는지

**예상되는 문제:**
- 스크롤이 너무 빠르거나 느릴 수 있음
- 답변 저장이 제대로 되지 않을 수 있음
- 역채점 로직이 제대로 작동하는지 확인 필요

---

### 2. **불필요한 파일 정리** (5분)
- [ ] `QuestionCard.tsx` 삭제 (더 이상 사용하지 않음)
- [ ] 사용하지 않는 import 정리

---

### 3. **에러 핸들링 기본 추가** (30분-1시간)

**필요한 에러 핸들링:**
- [ ] 답변 데이터 검증 (모든 문항 답변 여부)
- [ ] calculateScanResult 에러 처리
- [ ] 빈 답변 또는 잘못된 데이터 처리
- [ ] 네트워크 에러 처리 (향후 API 연동 시)

**예시:**
```typescript
try {
  const result = calculateScanResult(calculateAnswers, allQuestions);
  // 결과 표시
} catch (error) {
  // 에러 메시지 표시
  console.error('결과 계산 중 오류:', error);
}
```
