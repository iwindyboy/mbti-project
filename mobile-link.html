📋 접속 방법

  1. 모바일 기기와 PC가 같은 Wi-Fi 네트워크에 연결되어 있는지 확인하세요
  2. 모바일 기기의 브라우저에서 위 주소를 입력하거나
  3. QR 코드 스캐너 앱으로 링크를 스캔하세요
  4. 서버가 실행 중이어야 접속이 가능합니다

⚠️ 접속이 안 될 때 (방화벽 설정)

  1. Windows 방화벽 설정:
    • Windows 검색에서 "방화벽" 검색
    • "Windows Defender 방화벽" 열기
    • "고급 설정" 클릭
    • "인바운드 규칙" → "새 규칙" 클릭
    • 규칙 유형: "포트" 선택
    • 프로토콜: TCP, 특정 로컬 포트: 5173 입력
    • 작업: "연결 허용" 선택
    • 이름: "Vite Dev Server" 입력 후 완료
  2. 또는 PowerShell을 관리자 권한으로 실행 후:
    netsh advfirewall firewall add rule name="Vite Dev Server" dir=in action=allow protocol=TCP localport=5173
